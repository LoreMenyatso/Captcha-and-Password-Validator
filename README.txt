Description
This is a web application that is used to show the strength of a user's password and once the provided password and accepts passwords with complex characters. Furthermore, to ensure 
that the user is a human, a text-based captcha is provided for the user to solve. If successful the user will be granted access and redirected to the home index page.

How to run the application
Click on the login.html file located in the main directory of the application
